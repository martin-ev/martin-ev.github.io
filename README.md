# martin-ev.github.io

Personal academic website, live at <https://martin-ev.github.io>.

## Structure

| File | Purpose |
| --- | --- |
| [index.html](index.html) | Main page. |
| [styles.css](styles.css) | Layout, typography, light/dark theming |
| [icons.css](icons.css) | Inline SVG custom icon set |
| [diffusion.css](diffusion.css) | Styling for the animated background |
| [script.js](script.js) | Theme toggle, responsive nav / hamburger menu, scroll-reveal, email obfuscation. |
| [diffusion.js](diffusion.js) | Diffusion animation in the background |
| [404.html](404.html) | 404 not-found page |
| [home/index.html](home/index.html) | Redirects to the main page (old website URL) |
| [files/](files/) | Downloadable PDFs |
| [images/](images/) | Avatar and publication thumbnails |
| [robots.txt](robots.txt), [sitemap.xml](sitemap.xml) | Crawler hints |
